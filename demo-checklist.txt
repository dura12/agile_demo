Live demo checklist (GitHub Actions + password strength)

Before class
- Create GitHub repo (public), push this project to branch main.
- Run: npm install && npm test locally once (needs Node 18+).
- Optional: screenshot or short video of a green Actions run (backup if Wi-Fi fails).

Roles (split among group)
- Intro (30s): what the module does (score from length, digits, symbols).
- Screen driver: GitHub repo -> Actions tab -> latest CI run.
- Narrator: explain Checkout -> Setup Node -> npm install -> npm test.

During demo
- Expand each workflow step; say "same steps every push = continuous integration."
- Optional: push a tiny change (e.g. comment in test file), show second green run.
- Optional: break one assertion briefly, show red log, then revert (time permitting).

After
- Q&A: mention CD as "next step" (deploy job, secrets, environments).
